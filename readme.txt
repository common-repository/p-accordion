== Description ==

P-accordion plug-in to easily add multiple accordions in your WordPress website.

== Demo == 

https://davidebrunner.com/plung-in/p-accordion/#version

== Features ==

– 100% responsive.
– Easy use and installation.
- Graphic user interface configuration CSS.
- Space optimization and fast loading.
– [Shortcode] API function.
– High performances in your page.
– Accordion tags and accordion categorie tags.
– Easy to customize and developer friendly.
- Dinamic css change.

- R  shortcodes are interpreted.

- Q It is possible to disable mouse-over?
- R yes, inserting only # in the trans color.

== Simple set up  ==

Parameters:
title =”” text accordion.
titlecategorie=”” text title categorie.
addclass= “” add custom class title row.
panelclass= “” add custom class conteiner box.


=== Install ===

1. Upload the `p-accordion` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
